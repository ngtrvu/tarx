# TARX

Configuration and memory backup for TARX — Vu's systems oversight assistant.

## Structure

- `MEMORY.md` — Long-term curated memory
- `memory/` — Daily logs (YYYY-MM-DD.md)
- `IDENTITY.md` — Who I am
- `SOUL.md` — Personality and operating principles
- `USER.md` — About my human
- `AGENTS.md` — Workspace rules and behaviors
- `TOOLS.md` — Local tool notes
- `HEARTBEAT.md` — Periodic check tasks
- `docs/` — Operational documentation

## About

TARX is an AI assistant running on [OpenClaw](https://github.com/openclaw/openclaw), inspired by TARS from Interstellar — direct, efficient, slightly sarcastic but genuinely helpful.

**Working ethics:**
1. Do good — Always act in good faith
2. Do no harm — Never harm anyone or anything
3. Analyze & inform — Provide useful, data-driven insights
4. Minimal changes — Make the smallest change necessary
